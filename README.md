## How to run the app:
```
npm install
npm run createdb
npm run migratedb
```

**Make a `.env` file. Please use the [.env.example](.env.example) as the reference**

**Run this code:**
```
npm run dev
```
**Or,
Run these code if want the optimal app:**
```
npm run build
npm run start
```

**See the API documentation `http://localhost:3000/api-docs/`.**
